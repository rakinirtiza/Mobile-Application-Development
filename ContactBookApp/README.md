Contact Book App

This is a simple Android Contact Book application built using Kotlin and ListView.

## Features

- Search contacts in real-time
- Add new contact
- Delete contact (long press)
- Call button (shows toast)
- Show contact details (Toast)
- Dynamic avatar color
- Empty state view ("No contacts found")

## Technologies Used

- Kotlin
- Android Studio
- ListView
- Custom Adapter
- SharedPreferences

## Project Structure

- `MainActivity.kt` → Main logic
- `Contact.kt` → Data model
- `ContactAdapter.kt` → Custom adapter
- `activity_main.xml` → Main UI
- `item_contact.xml` → Contact row UI
