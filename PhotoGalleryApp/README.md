# Photo Gallery App

A simple and clean Android Photo Gallery App built using **Kotlin** and **GridView**.



# Features

- Display images in GridView (2 columns)
-  Fullscreen image view with zoom
- Category filtering (All, Nature, City, Animals, Food, Travel)
-  Multi-selection using long press
-  Delete selected images
-  Share selected images (Intent)
-  Add new photo using Floating Action Button (FAB)
- Clean and user-friendly UI


## Tech Stack

- Language: Kotlin
- UI: XML
- Components: GridView, BaseAdapter, Intent



# Project Structure

- `MainActivity.kt` → Main logic & UI handling  
- `FullscreenActivity.kt` → Full image view with zoom  
- `PhotoAdapter.kt` → Custom adapter for GridView  
- `Photo.kt` → Data model  
- `activity_main.xml` → Main UI  
- `item_photo.xml` → Grid item layout
