# Grade Report App

This is an Android application developed using Kotlin & lab task 4

## Features
- Display student information
- Show subject-wise marks in a table
- Calculate grade automatically
- Calculate GPA
- Add new subjects dynamically
- Show total, passed, and failed subjects
- Share report via other apps

## Technologies
- Kotlin
- Android Studio
- TableLayout

## Description
This app is developed as part of a Mobile Application Development lab task. It allows users to manage student grades and view GPA easily.
