# News Article Reader App

## Project Overview
This is a News Article Reader Android application developed using Kotlin. The app displays a long-form news article with a clean and user-friendly UI. Users can scroll through content, navigate between sections, bookmark the article, and share it.

# Features
-  Display full news article
-  Sticky toolbar with title, bookmark & share buttons
- Header image with category tag
-  Author section (name & date)
-  Quick navigation buttons (Introduction, Key Points, Analysis, Conclusion)
-  Bookmark toggle with Toast message
-  Share article using Android Intent
-  Tags section
-  Related articles with images
-  Back to top button

# Technologies Used
- Kotlin
- XML Layout
- Android Studio
- NestedScrollView
- Intent (for sharing)
